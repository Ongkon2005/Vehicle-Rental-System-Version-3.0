# 🚗 Vehicle Rental System

A desktop-based Vehicle Rental System developed using Java Swing that allows users to browse available vehicles, make bookings, calculate rental costs, generate payment slips, and manage rental information through an interactive graphical interface.

---

## 📖 Project Description

The Vehicle Rental System is designed to simplify the process of renting vehicles. Users can select different types of vehicles, specify rental duration, calculate costs automatically, and complete payments through an easy-to-use interface.

The system follows Object-Oriented Programming (OOP) principles and provides a practical implementation of inheritance, encapsulation, polymorphism, and GUI development using Java Swing.

## 📂 Main Components

### Vehicle Class
Base class containing common vehicle information.

### Car Class
Represents car rental information.

### Bike Class
Represents bike rental information.

### Truck Class
Represents truck rental information.

### Booking Class
Handles booking operations and rental details.

### Billing Class
Calculates rental charges and generates bills.

### Payment Dialog
Handles payment-related operations.

### Payment Slip
Generates payment receipts for customers.

---

## 🎯 Object-Oriented Concepts Used

### Encapsulation
Private attributes with public getter and setter methods.

### Inheritance
Car, Bike, and Truck classes inherit from the Vehicle class.

### Polymorphism
Different vehicle types implement their own behavior.

### Abstraction
Common functionalities are defined in parent classes.
